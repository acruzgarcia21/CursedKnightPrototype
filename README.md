# Cursed Knight

Cursed Knight is a dark fantasy deck-building game developed in Unity. Fight through cursed enemies, manage limited resources, and harness forbidden corruption to survive increasingly dangerous encounters.

## Overview

Built as a solo development project, Cursed Knight combines strategic turn-based combat with risk-reward decision making. Players must carefully balance defense, offense, and corruption while battling through hostile encounters.

## Current Features

### Combat System

* Turn-based card combat
* Energy-based card play system
* Attack, Defense, and Utility cards
* Enemy AI and attack patterns
* Block and damage mitigation
* Win and loss conditions

### Deck Systems

* Draw pile
* Discard pile
* Automatic deck reshuffling
* Hand management
* ScriptableObject-based card architecture

### Corruption System

* Cards can generate corruption
* Corruption is tracked throughout combat
* Curse Surge triggers when corruption reaches its limit
* Risk-versus-reward gameplay centered around corruption management

### Enemies

Current prototype includes multiple enemy types with unique combat behaviors:

* Crypt Rat
* Bone Soldier
* Hollow Brute
* Chainbound Knight (Boss)
